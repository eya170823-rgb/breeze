# 브리즈부동산중개 BREEZE 홈페이지

매물(구글시트)·뉴스/게시판(네이버 블로그)이 **자동으로 연동**됩니다.

## ⚡ 빠른 요약
- **매물 올리기**: 구글시트 `등록자` 칸에 `광고완료`
- **뉴스/게시판**: 네이버 블로그 부동산뉴스/게시판 카테고리에 글
- **간단 수정**: `config.js`의 따옴표 안 글자 (전화·주소·영업시간·사업자번호 등)
- **개인정보**: 시트 비공개로 둬도 OK (엔진이 안전한 정보만 전달)

## 기술
순수 HTML/CSS/JS + 구글 앱스 스크립트 엔진. 빌드 과정 없음.
