# BREEZE 브리즈부동산중개 홈페이지

> 바른 기준, 가치 있는 동행 — BREEZE가 당신과 함께합니다.

제주시 연동 **브리즈부동산중개 BREEZE**의 모바일 우선 인터랙티브 홈페이지입니다.
순수 HTML/CSS/JS로 만들어 별도 서버나 빌드 과정 없이 동작합니다.

## 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 페이지 구조 (히어로 · 매물 · 소식 · CTA · 오시는길) |
| `styles.css` | 디자인 (퓨어 화이트 / 그레이 / 뮤트 테라코타) |
| `config.js` | **여기만 수정** — 전화번호·주소·구글시트 ID |
| `sheets.js` | 구글 시트 → 매물/소식 데이터 변환 |
| `sample-data.js` | 시트 연결 전 보여줄 샘플 매물 |
| `app.js` | 렌더링·필터·검색·모달 등 동작 |

## 1. 바로 미리보기

`index.html` 을 더블클릭하면 브라우저에서 바로 열립니다.
(검색·필터·전화 버튼·지도까지 모두 동작합니다. 매물은 샘플이 표시됩니다.)

## 2. 내 구글 시트 연결하기 ⭐

1. 매물/소식이 들어있는 **구글 시트**를 엽니다.
2. 시트 하단 탭 이름을 **`매물`**, **`소식`** 으로 맞춥니다.
3. **매물** 탭 첫 행(머리글)을 아래 순서로 만듭니다:
   ```
   매물종류 | 거래유형 | 제목 | 가격 | 위치 | 면적 | 층 | 방향 | 설명 | 이미지URL | 상태 | 등록일
   ```
   - `가격`: 숫자만 넣으면(만원 단위) 자동으로 "억/만원"으로 표시됩니다. 예) `68000` → `6억 8,000만원`. `보증 5000/월 350` 처럼 글자로 넣어도 됩니다.
   - `상태`: 비우거나 `공개` = 노출, `숨김`/`비공개` = 숨김
   - `이미지URL`: 비워도 됩니다(BREEZE 기본 썸네일 표시)
4. **소식** 탭 첫 행:
   ```
   제목 | 내용 | 카테고리 | 날짜 | 링크 | 이미지URL
   ```
5. 구글 시트 우측 상단 **[공유] → "링크가 있는 모든 사용자: 뷰어"** 로 설정합니다.
6. 시트 주소창에서 ID를 복사합니다:
   `https://docs.google.com/spreadsheets/d/`**`여기가 ID`**`/edit`
7. `config.js` 의 `sheetId: ""` 안에 붙여넣습니다.

이제 **시트만 수정하면 홈페이지 매물이 자동으로 바뀝니다.** (새로고침 시 반영)

## 3. 전화번호·주소 바꾸기

`config.js` 에서 `phone`, `address` 값을 수정하면 사이트 전체에 반영됩니다.
