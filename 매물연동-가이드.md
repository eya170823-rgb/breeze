# 매물 연동 가이드 (오픈 준비되면 진행)

> 지금은 **샘플 매물**만 표시됩니다. 아래는 **실제 매물을 띄울 준비가 됐을 때** 진행하는 절차예요.
> 핵심: 시트는 **비공개로 유지**하고, **캘린더 칸에 표시한 매물만** 개인정보 없이 홈페이지에 올라갑니다.

## 0. 준비 개념
- 표시할 매물의 **`캘린더` 칸에 `O`** (또는 `홈페이지`) 라고 적습니다. → 그 매물만 공개.
- 소유자·임차인·의뢰인 전화, 비공개메모, 주소 등 **개인정보는 절대 안 나갑니다.** (스크립트가 안전한 열만 골라 보냄)

## 1. 앱스 스크립트 설치
1. 구글 시트 상단 **확장 프로그램 → Apps Script** 클릭
2. 기본 코드(`function myFunction(){}`)를 모두 지우고,
   `apps-script/매물피드.gs` 파일 내용을 **전부 복사해서 붙여넣기**
3. 맨 위 `LISTING_TABS` 의 탭 이름이 **실제 시트 탭 이름과 똑같은지** 확인
   (다르면 그 탭은 그냥 건너뜁니다 — 오류 아님)
4. 💾 저장(Ctrl+S)

## 2. 웹앱으로 배포
1. 오른쪽 위 **배포 → 새 배포**
2. 톱니바퀴 → **웹 앱** 선택
3. 설정:
   - 실행 사용자: **나(본인)**  ← 이게 있어야 시트를 비공개로 둬도 됩니다
   - 액세스 권한: **모든 사용자**
4. **배포** → 처음이면 **권한 승인**(본인 구글 계정 → 고급 → 안전하지 않음으로 이동 → 허용)
5. 나오는 **웹 앱 URL**(`.../exec` 로 끝남)을 복사

## 3. 홈페이지에 연결
1. `config.js` 의 `feedUrl: ""` 안에 복사한 URL 붙여넣기
2. 저장 → 홈페이지 새로고침 → **캘린더에 표시한 매물**이 매물 페이지에 뜹니다 🎉

## 4. 평소 사용
- 매물 올리기: 해당 행 **캘린더 칸에 `O`** → 자동 노출
- 내리기: 캘린더 칸 **비우기** → 자동 사라짐
- 시트 내용(가격·설명 등)을 고치면 홈페이지도 자동 반영(새로고침 시)

## 사진은?
- `사진` 칸에 **웹 주소(http로 시작하는 이미지 링크)**가 있으면 자동 표시됩니다.
- 앱시트로 올린 사진은 드라이브에 저장돼 바로는 안 보일 수 있어요. 사진 한 장 올려보시고 알려주시면, 그 형식에 맞춰 표시 방법을 맞춰 드리겠습니다.
- 사진이 없으면 **BREEZE 기본 썸네일**로 깔끔하게 표시됩니다.
