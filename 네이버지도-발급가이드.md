# 네이버 지도 인증키(Client ID) 발급 가이드

페이지 안에 **실제 네이버 지도**를 띄우려면 무료 인증키가 필요합니다.
아래 순서대로 5~10분이면 발급됩니다. 발급한 **Client ID**만 `config.js`에 넣으면 끝입니다.

---

## 1. 네이버 클라우드 플랫폼 가입
1. https://www.ncloud.com 접속 → 우측 상단 **회원가입**(또는 로그인)
2. 본인인증(휴대폰) + **결제수단(카드) 등록**이 필요합니다.
   - 지도는 **월 무료 사용량이 넉넉**해서, 일반 홈페이지 방문량으로는 요금이 거의 발생하지 않습니다.

## 2. Maps 서비스 신청
1. 로그인 후 우측 상단 **콘솔(Console)** 클릭
2. 상단 검색창에 **Maps** 입력 → **Maps** 서비스 선택
3. **이용 신청하기** 클릭

## 3. Application(앱) 등록
1. **Application 등록** 버튼 클릭
2. **Application 이름**: 아무거나 (예: `breeze-homepage`)
3. **Service 선택**: **Web Dynamic Map**(웹 동적 지도) 체크 ✅
4. **Web 서비스 URL** 등록 — ⭐ 매우 중요 (여기 등록한 주소에서만 지도가 떠요)
   아래 주소들을 **각각 추가**해 주세요:
   - `http://localhost:8000`  (PC 테스트용)
   - `http://172.30.1.86:8000`  (폰 테스트용)
   - 나중에 인터넷에 올리면: `https://본인아이디.github.io`  (GitHub Pages 주소)
5. **등록** 완료

## 4. Client ID 복사
1. 등록된 Application 목록에서 방금 만든 앱의 **인증 정보**(Authentication) 확인
2. **Client ID** 값을 복사 (영문+숫자 조합)

## 5. config.js에 붙여넣기
`config.js` 파일에서 이 부분을 찾아 Client ID를 넣습니다:

```js
naver: {
  mapClientId: "여기에_복사한_ClientID",   // ← 붙여넣기
  lat: 33.4889,
  lng: 126.4914,
  ...
}
```

저장 후 **http://localhost:8000** 에서 새로고침하면 푸터에 실제 네이버 지도가 뜹니다.

---

## ⚠️ 자주 묻는 것
- **지도가 안 떠요 / 빈칸이에요** → 지금 보는 주소가 4번에서 등록한 URL과 같은지 확인하세요.
  특히 파일을 더블클릭(`file:///...`)하면 안 됩니다. 반드시 `http://localhost:8000` 으로 접속.
- **핀 위치가 살짝 달라요** → `config.js`의 `lat`/`lng` 숫자를 조정하면 됩니다.
  정확한 좌표는 네이버 지도에서 사무실 위치를 길게 눌러 확인할 수 있어요. (원하시면 맞춰 드립니다)
- **Client ID는 외부에 노출돼도 되나요?** → 네이버 지도 Client ID는 원래 화면(프론트엔드)에 노출되는 값이고,
  4번의 URL 제한으로 보호되므로 괜찮습니다.

발급 중 막히는 화면이 있으면 그 화면을 알려주세요. 바로 도와드리겠습니다.
