# 홈페이지 엔진 설치 가이드 (자동화)

이걸 한 번 설치하면 — **시트는 비공개로 두고도** 매물·뉴스·게시판이 홈페이지에 뜨고,
**등록일은 매일 아침 자동 갱신**됩니다. 버튼·프로그램 없이 거의 다 자동이에요.

> 핵심: 매물은 **`등록자` 칸에 `광고완료`** 적은 것만 공개. 계약완료는 회색, 거래완료는 자동 숨김.

---

## 1. 스크립트 설치
1. 구글 시트 상단 **확장 프로그램 → Apps Script**
2. 기존 코드를 모두 지우고, `apps-script/홈페이지엔진.gs` **내용 전체 복사 → 붙여넣기**
3. 위쪽 설정값 확인 (보통 그대로 두면 됨):
   - `LISTING_TABS` = 매물 탭 이름들 (실제 탭과 같아야 함)
   - `NEWS_CATEGORY` / `BOARD_CATEGORY` = 블로그 카테고리 이름
4. 💾 저장 (Ctrl+S)

## 2. 웹앱으로 배포
1. 오른쪽 위 **배포 → 새 배포**
2. 톱니바퀴 ⚙ → **웹 앱**
3. 설정:
   - 실행 사용자: **나(본인)**  ← 시트를 비공개로 둬도 되는 이유
   - 액세스 권한: **모든 사용자**
4. **배포** → 처음이면 **권한 승인** (내 계정 → 고급 → "(프로젝트)로 이동(안전하지 않음)" → 허용)
5. 나온 **웹 앱 URL**(`...macros/s/..../exec`)을 복사

## 3. 홈페이지에 연결
1. `config.js` 의 `feedUrl: ""` 안에 복사한 URL 붙여넣기
2. 같은 파일 `sheetId` 는 **비워두기** (`sheetId: ""`) — 이제 엔진으로 받으니까
3. 저장 → 홈페이지 새로고침 → 광고완료 매물이 뜹니다 🎉
4. 이제 **구글시트 공유를 '비공개'로 되돌려도** 홈페이지는 잘 동작합니다 (개인정보 안전)

## 4. 자동 갱신 예약 (한 번만)
1. Apps Script 편집기 상단 함수 목록에서 **`installDailyTriggers`** 선택 → **실행 ▶**
2. (권한 한 번 더 물으면 허용)
3. → 이제 **매일 아침 7시에 광고완료 매물의 등록일이 자동으로 오늘 날짜**가 됩니다.
   - 손 안 대도 매물이 매일 신선하게 유지돼요. (시간 바꾸려면 코드의 `atHour(7)` 숫자 변경)

---

## 평소 사용 (이게 전부예요)
- **매물 올리기**: 해당 행 `등록자` 칸에 **`광고완료`** → 자동 노출
- **내리기**: `광고완료` 지우기 → 자동 사라짐
- **계약/거래 상태**: `구분`(또는 `상태`)에 계약완료/거래완료 적으면 → 회색 표시 / 자동 숨김
- **뉴스·게시판**: 블로그 해당 카테고리에 글 올리면 → 자동 표시
- **등록일**: 매일 자동 (4번 예약 덕분)

## 블로그 카테고리 주의
네이버 RSS는 **상위 카테고리**만 알려줍니다. 그래서 `부동산뉴스`·`게시판`이 다른 카테고리 **하위**에 있으면 인식이 안 될 수 있어요.
→ 두 카테고리를 **상위(독립) 카테고리**로 두면 가장 확실합니다. (안 되면 엔진의 `NEWS_CATEGORY` 값을 실제로 RSS에 잡히는 이름으로 바꾸면 됩니다.)

막히는 화면 있으면 캡처해서 알려주세요. 같이 진행합니다.
