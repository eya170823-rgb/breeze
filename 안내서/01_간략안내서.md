# BREEZE 홈페이지 — 간략 안내서

> 브리즈부동산중개 BREEZE 홈페이지를 **빠르게 이해하고 운영**하기 위한 1장 요약.
> 자세한 내용은 [02_상세안내서.md](02_상세안내서.md) 참고.

---

## 1. 한 줄 요약
- **매물·뉴스·게시판이 자동으로 채워지는** 부동산 홈페이지.
- 사장님은 **구글시트**(매물)와 **네이버 블로그**(뉴스·게시판)만 평소대로 쓰면 됨. → 홈페이지에 **자동 반영**.

## 2. 메뉴 4개
| 메뉴 | 내용 |
|---|---|
| **홈** | 소개·특징·상담 안내 |
| **매물** | 종류/지역/거래/가격 필터로 매물 검색 |
| **뉴스** | 블로그 '부동산뉴스' 글 자동 표시 |
| **게시판** | 블로그 '게시판' 글 자동 표시 |

## 3. 매물 올리기 (제일 중요)
구글시트 매물장에서:
1. 올릴 매물 행의 **`등록자` 칸에 `광고완료`** 입력 → **홈페이지에 자동 노출**
2. 내리려면 → `광고완료` 글자 **지우기**
3. 상태: `구분`(또는 `상태`)에
   - **공실** → 정상 표시
   - **계약완료 / 임대중** → 사진 회색 + 상태 표시
   - **거래완료** → 자동으로 사라짐

> 개인정보(소유자·임차인 전화, 메모 등)는 **절대 홈페이지에 안 나갑니다.** 안심.

## 4. 뉴스·게시판 올리기
- 네이버 블로그 **부동산뉴스** 카테고리에 글 → **뉴스**에 자동
- **게시판** 카테고리에 글 → **게시판**에 자동
- (카테고리는 블로그 **상위 카테고리**로 둬야 인식됨)

## 5. 색·글씨·연락처
- **색**: 화이트 배경 + 그레이 + **테라코타(붉은 갈색)** 포인트
- **글씨체**: Pretendard (깔끔한 고딕)
- **전화** 010-2601-0110 · **주소** 제주시 연동 260-18

## 6. 사소한 글자 수정 (직접 가능)
- `config.js` 파일에서 따옴표 `" "` 안의 글자만 바꾸면 됨 (영업시간·사업자번호·주소 등)
- 인터넷 공개 후엔 **github.com에서 ✏️ 연필 → 수정 → 저장** 하면 1~2분 후 반영
- 헷갈리면 **그냥 담당자에게 요청**해도 됨

## 7. 절대 주의 ⚠️
- 구글시트의 **탭(시트) 이름·열 이름 함부로 바꾸지 말 것** (앱시트·홈페이지 둘 다 깨질 수 있음)
- 홈페이지 엔진은 **앱시트·캘린더 코드와 별개**라 서로 안 건드림
- 시트는 **비공개로 둬도** 홈페이지 정상 작동 (개인정보 보호)
